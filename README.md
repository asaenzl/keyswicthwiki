# Keyboard Switch Wiki

A comprehensive wiki showcasing the different mechanical keyboard switches available on the market. Explore and compare various switch types to find the perfect one for your typing experience.

## About

This project provides detailed information about mechanical keyboard switches, helping enthusiasts and newcomers alike discover the diverse range of switches available. Whether you're looking for tactile feedback, smooth linear action, or satisfying clicky switches, you'll find them all catalogued here.
